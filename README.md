# 📦 Import Tracker — Облік імпортного товару

## Що це
Веб-програма для обліку отримання товарів з-за кордону.
Всі твої угоди, контейнери, дати, фінанси та товари — в одному місці.

## Як запустити (один раз)

### 1. Встанови Python (якщо ще не встановлений)
Завантаж з https://python.org — обирай версію 3.10+

### 2. Встанови Flask
Відкрий термінал (командний рядок) у папці з програмою і виконай:
```
pip install flask
```

### 3. Запусти програму
```
python app.py
```

### 4. Відкрий браузер
Перейди на: **http://127.0.0.1:5000**

---

## Що вміє програма

### Активні угоди
- Таблиця всіх активних угод
- Підсвітка: 🔴 прострочено / 🟡 прибуває за 7 днів / 🔵 в дорозі
- Пошук по номеру угоди, постачальнику, контейнеру

### Кожна угода містить
- Номер угоди, контейнери, умови (FOB/CIF тощо)
- ETD та ETA з автоматичним підрахунком днів до прибуття
- Маршрут: POL → POD, лінія
- Фінанси: фрах + наземка = загальна сума, % оплати, залишок
- Товари: назва, кількість, одиниця, ціна (кілька позицій)
- Документи: відмітка про отримання

### Архів
- Кнопка 📦 відправляє завершену угоду в архів
- Архів доступний окремо, легко шукати
- Можна відновити угоду з архіву
